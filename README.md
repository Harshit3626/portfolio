# Harshit Srivastava — Portfolio

A single-file, interactive client-acquisition site for freelance and contract work across **data engineering, ML, AI agents, analytics, automation, and web development**.

🔗 **Live:** [harshit-data-ml-automation-engineer.vercel.app](https://harshit-data-ml-automation-engineer.vercel.app)

## About

Built around a live telemetry / AIOps aesthetic — the page behaves like a working system, not a static brochure. Visitors can see the full range of services at a glance, browse proof-of-work, and send a pre-filled project brief in a couple of taps.

### Features

- **Live telemetry hero** — animated metrics and self-updating sparklines
- **Interactive service selector** — tap services to build a brief; selections flow into the contact form
- **Proof of work** — shipped, open-source projects with result-first framing
- **Multiple contact paths** — pre-filled brief form, email, WhatsApp, and LinkedIn
- **Fully responsive** — works down to mobile, with reduced-motion support and keyboard focus

## Tech

Plain `HTML`, `CSS`, and vanilla `JavaScript` — zero dependencies, zero build step. Just one `index.html`.

## Run locally

Open `index.html` in any browser. That's it.

## Deploy

Hosted on [Vercel](https://vercel.com). Any push to `main` auto-deploys once the repo is connected.
